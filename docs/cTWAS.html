Methods

We have run the multigroup branch of cTWAS on each cell type seperately. Three time points serve as the “group” for the cell type run.

eGenes for each condition (the combination of cell type and time point) was selected with three criteria: 1. At least one SNP-gene pair has FDR<0.05 in any time point within the same cell type; 2. P value < 0.1 for the current condition. The number of genes for different settings were shown below; 3. Must be a protein coding gene

Below are the eGenes selected for imputation in cTWAS. Multigroup cTWAS contain much more eGenes due to constructing a union.

Ocurrence of region merging

cTWAS will automatically merge LD regions if a gene spans across multiple regions. This is computationally expensive. Fortunately, with the merging option turned on, no merging was performed. This is probably because none of the genes selected for multi-group cTWAS spanned multiple LD regions.

Rerunning cTWAS on subset fo genes

Since PIP was estimated with 10% thining, we need to esstimate the PIP and performed fine mapping on the top genes. Specifically, with the same imputed gene expression and group priors, we run cTWAS at the specific gene locations with the full set of SNPs.

PIP comparison between initial run and the rerun

Most top gene PIPs were overestimated in the initial run. There are 17 genes with PIP sums greater than 0.8 for the GABA cell type. Only 2 genes cross the threshold in both Glut cell types. In addition, all the remaining top genes in the Glut cell types were also in the top gene list of GABA. The NPIPB2 has high PIP in all cell types.
